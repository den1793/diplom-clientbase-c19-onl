# Project name

Project Clientbase program

# Description

Web application that allows manage projects in various business.

## Dependencies

* Spring Boot
* Spring Security
* Java Servlet API
* Spring ORM
* Hibernate Core
* PostgreSQL
* Tomcat DBCP
* Hibernate Validator Engine
* Java Mail Sender
* Mapstruct
* Thymeleaf
* slf4j + log4j
* JUnit 5
* String Test
* Lombok


# Installation

## Requirements

* JDK 11
* Maven
* PostgreSQL
* Apache Tomcat 9
